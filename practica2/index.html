Práctica 2

Soldadura

En esta práctica aprendimos a soldar componentes electrónicos en una placa PCB, aplicando las técnicas adecuadas para asegurar un buen contacto y funcionamiento del circuito.

Objetivo: Comprender y aplicar las técnicas adecuadas de soldadura en componentes electrónicos, garantizando conexiones funcionales y de calidad en las placas de circuito impreso (PCB).

Información general

¿Qué es?

La soldadura de componentes electrónicos es el proceso mediante el cual se unen los terminales de los componentes a las pistas de una placa de circuito impreso (PCB) utilizando un material metálico llamado estaño. Este material se funde con la ayuda de un cautín y, al enfriarse, forma una conexión eléctrica y mecánica firme. Su objetivo es asegurar que los componentes queden bien sujetos y que la corriente eléctrica pueda fluir correctamente entre ellos, permitiendo el funcionamiento adecuado del circuito.

1

Instalación de Librerías FabLab

El primer paso fue descargar e instalar la librería del FabLab en KiCad. Esta librería contiene componentes específicos optimizados para fabricación digital, con las especificaciones y footprints adecuados para el proceso de fresado que utilizaríamos posteriormente.

Es crucial usar componentes de la librería FabLab ya que están diseñados específicamente para el proceso de fabricación que usaremos.
2

Creación del Esquemático

Cada integrante del equipo diseñó su propio circuito. La mayoría incluyó:

  • 4 interruptores (switches) con salida
  • Voltaje de alimentación: 3.3V
  • 8 resistencias para limitación de corriente
  • 4 LEDs como indicadores
  • Conexión a tierra (GND)
Excepción: El circuito de Iván incluyó 14 resistencias en su diseño para una configuración más compleja.
3

Asignación de Huellas

Después de completar el esquemático, asignamos las huellas correspondientes a cada componente. Las huellas definen la forma física y las dimensiones de soldadura de cada componente en la PCB.

4

Diseño del Layout de PCB

Pasamos al editor de PCB donde realizamos el ruteo de las conexiones. Este proceso incluye:

  • Colocación estratégica de componentes
  • Ruteo de pistas evitando cruces
  • Definición del contorno de la placa
  • Verificación de reglas de diseño (DRC)
5

Generación de Archivos Gerber

Finalmente, exportamos los archivos necesarios para la fabricación en formato PNG de alta resolución, incluyendo las capas de pistas de cobre y el contorno de corte de la placa.

Los archivos PNG generados son compatibles con el software MODs para la fabricación.

Proceso de Soldadura

¿Qué es MODs?

MODs (modsproject.org) es una herramienta web modular que convierte archivos de diseño en códigos de control para máquinas de fabricación digital, como fresadoras CNC.

1

Carga de Archivos PNG

Importamos los archivos PNG generados en KiCad al entorno web de MODs. Cargamos tanto el archivo de pistas como el de contorno de la placa, cada uno requiere configuraciones específicas de fresado.

2

Configuración de Parámetros de Fresado

Configuramos los parámetros esenciales para el proceso:

  • Tipo de fresa: 1/64" para pistas, 1/32" para contorno
  • Profundidad de corte: 0.1mm para pistas
  • Velocidad de avance: Optimizada para cobre
  • Velocidad del husillo: Según especificaciones de MonoFab
3

Configuración de Módulos

MODs funciona mediante módulos interconectados. Configuramos la cadena de procesamiento:

  • Entrada: Módulo de lectura de imagen PNG
  • Procesamiento: Módulo de cálculo de trayectorias
  • Salida: Módulo de generación de G-code
4

Generación de Trayectorias

El software calcula automáticamente las trayectorias de la fresa, optimizando el recorrido para minimizar el tiempo de mecanizado y evitar dañar las pistas ya fresadas.

Es importante revisar visualmente las trayectorias antes de proceder al fresado para detectar posibles errores.
5

Exportación de G-code

Finalmente, exportamos el código G que controla los movimientos de la fresadora MonoFab. Este archivo contiene todas las instrucciones necesarias para fabricar la PCB físicamente.

El archivo G-code está listo para ser cargado en la fresadora MonoFab.

Proceso en MonoFab

1

Preparación del Material

Colocamos la placa virgen de cobre en la fresadora, asegurándonos de que esté perfectamente nivelada y fijada para evitar movimientos durante el fresado. La preparación correcta del material es crucial para obtener resultados precisos.

2

Instalación y Calibración de Fresas

Instalamos primero la fresa de 1/64" para el fresado de pistas. Calibramos la altura de la herramienta utilizando el sistema de auto-nivelación de la MonoFab, asegurando un contacto preciso con el material.

3

Fresado de Pistas

Ejecutamos el primer archivo G-code para fresar las pistas del circuito. La máquina retira el cobre de la superficie, dejando definidas las conexiones eléctricas según el diseño digital. Este proceso requiere precisión para mantener la continuidad de las pistas.

4

Cambio de Fresa y Corte de Contorno

Cambiamos a la fresa de 1/32" y ejecutamos el segundo archivo para cortar el contorno de la placa. Esto define la forma final de la PCB y la separa del material base.

5

Inspección y Acabado

Retiramos la placa terminada y realizamos una inspección visual para verificar que las pistas estén correctamente delineadas y sin imperfecciones. Limpiamos los residuos de cobre y preparamos la PCB para el proceso de soldadura de componentes.

¡PCB fabricada exitosamente! El diseño digital se ha materializado en un prototipo físico funcional.

Lecciones Aprendidas

  • Preparación adecuada: Tener a la mano todas las herramientas y componentes facilita un proceso de soldado más ordenado y seguro.
  • Temperatura correcta: Ajustar bien la temperatura del cautín evita dañar los componentes o las pistas de la PCB.
  • Aplicación precisa del estaño: Usar la cantidad justa de estaño garantiza una buena conexión sin provocar cortocircuitos.
  • Seguridad y limpieza: Mantener el área de trabajo limpia y usar protección adecuada previene accidentes y mejora la calidad del resultado.
  • Revisión final: Inspeccionar cada soldadura ayuda a detectar uniones frías o defectuosas antes de probar el circuito.
